### BIG O NOTATION

#Array

| Type   | Time Complexity |
|--------|-----------------|
| Lookup | O(1)            |
| Insert | O(n)            |
| Delete | O(n)            |


#linked list

| Type             | Time Complexity |
|------------------|-----------------|
| Lookup by value  | O(n)            |
| Lookup by index  | O(n)            |
| Insert end       | O(1)            |
| Insert beggining | O(1)            |
| Insert Middle    | O(n)            |
| Delete end       | O(n)            |
| Delete beggining | O(1)            |
| Delete Middle    | O(n)            |
